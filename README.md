# Calculadora Pro Interactiva - Módulo 3

## Descripción
Este proyecto es una aplicación de calculadora híbrida que funciona tanto en una interfaz web visual como mediante un modo interactivo de consola/prompts. Desarrollada para la evaluación del Módulo 3 de Programación en JavaScript.

## Tecnologías Utilizadas
- **HTML5 / CSS3** (Interfaz y estilos modernos)
- **JavaScript (ES6+)**:
  - Objetos para gestión de estado y métodos.
  - Arreglos y método `.map()` para el historial.
  - Estructuras de control `while`, `switch` e `if`.
  - Funciones de flecha modularizadas.

## Cómo ejecutar
1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en cualquier navegador moderno.
3. Puedes usar los botones en pantalla o presionar el botón azul **"Activar Modo Interactivo"** para iniciar el flujo de prompts.
